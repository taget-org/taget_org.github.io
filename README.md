# TAGET - Tanzanite Generation Tracers

Official website ya TAGET (Tanzanite Generation Tracers).

## Kuhusu TAGET

TAGET ni taasisi ya vijana iliyoanzishwa mwaka 2023 kwa lengo la kuamsha, kuhamasisha, na kuunda jamii ya vijana wenye dira, uthubutu na ubunifu.

## Malengo Yetu

- Kutoa ajira kwa vijana
- Kuhamasisha sayansi na teknolojia
- Kutatua changamoto za kijamii
- Kuhimiza amani na mshikamano
- Kukuza vipaji vya vijana
